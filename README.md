# Cube Wireframe Animation

Tiny demo that draws and rotates a wireframe cube on a 2D canvas.

## Run
- Open index.html in a browser; the cube should rotate automatically.
- If the canvas is blank, ensure the script runs after the canvas tag (add `defer` to the script or move it to the end of body).

## Add a GIF preview
1. Export your animation GIF to cube_it/media/cube.gif.
2. Embed it in this README:

```md
![Rotating cube](media/cube.gif)
```

Or show it on the page by adding below the canvas in index.html:

```html
<img src="media/cube.gif" alt="Rotating cube" width="240" />
```
